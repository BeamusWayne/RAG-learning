# 环境与依赖说明

与机试说明页「通用前置条件」一致，建议候选人使用以下环境完成机试。

## 基础环境

- **Python**：3.12+
- **系统**：Windows / macOS / Linux 均可

## 依赖（按场景）

| 用途 | 依赖 | 说明 |
|------|------|------|
| Agent 框架 | LangChain、LangGraph | 搭建 RAG、ReAct、工具调用与流程编排 |
| 数据分析（场景三） | Pandas | 日运营 CSV 的加载、清洗与统计 |
| 记忆（场景二可选） | 内存/本地存储 | 机试环境无 Redis；短期记忆、历史执行记录请用内存或本地文件等实现 |
| 大模型 | OpenAI API / Qwen 等 | 本地轻量或云端 API，按考场要求配置 |
| 向量检索（场景四可选） | 本地向量库或 mock | 素材库 RAG 可先用 materials 提供的 mock 检索 |
| LLM模型 | ollama | 免费模型，使用地址： https://ollama.sjzt2020.com |

所有机试资料中的工具与数据均为模拟，不依赖真实游戏后台或外网 API。
